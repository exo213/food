# Food Randomizer (React + Vite)
A premium, glassmorphic food randomizer application built with React, Vite, and Tailwind CSS.

![Project Preview](https://placehold.co/600x400/1e293b/ffffff?text=Food+Randomizer)

## Tech Stack
- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS 3
- **Icons**: Lucide React
- **Animations**: Framer Motion

## Getting Started

### Prerequisites
- Node.js installed

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`).

### Building for Production
To create a production build:
```bash
npm run build
```
The output will be in the `dist` folder.

## Features
- **Glassmorphism UI**: Modern aesthetic with blur effects.
- **Slot Machine Animation**: Tactile feeling when selecting a meal.
- **Responsiveness**: Works on all screen sizes.
- **Preserved Data**: Uses the original food lists suitable for Breakfast, Lunch, and Dinner.
